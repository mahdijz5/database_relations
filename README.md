## Description

> This is good example for understanding relations.
> <br>
> This project contains all type of relations beside **tree structure** and **junction table** 



## Installation

```bash
$ npm install
```

## Running the app

```bash
# to create tables
$ npm run start

# to seed data
$ npm run seed:run

# to start development mode
$ npm run start:dev
```
